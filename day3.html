Q.create a box and increase its width on hover , add transition on hover 

  •           HTML Code:-
          <div class="container2">
            <div class="box p2box1">Only width increase</div>
            <div class="box p2box5">Ease-In</div>
          </div>

          CSS Code:-
          .container2 {
            display: flex;
            flex-direction: column;
            align-items: center;
          }
          .p2box1 {
            width: 200px;
            height: 200px;
            background-color: aqua;
            transition: width 2s;
            margin-bottom: 100px;
            margin-top: 100px;
          }
          .p2box1:hover {
            width: 400px;
          }
          .p2box5 {
            width: 200px;
            height: 200px;
            background-color: blue;
            transition: width 2s;
          }
          .p2box5:hover {
            width: 400px;
            transition-timing-function: ease-in;
          }
